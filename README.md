# Portfolio — Aakash Deep Yadav

Personal portfolio website built with React, TypeScript, and Tailwind CSS.

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS v4
- Framer Motion
- Lucide React

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
